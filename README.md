# Awesome Daily 素材库

初中课内同步深度学习素材集 · 与人教版七年级下册（语文 / 历史 / 道德与法治）匹配。

## 内容（37 篇）

```
语文/
├── 文言文/  6 篇（孙权劝学、卖油翁、陋室铭、爱莲说、活板、木兰诗）
└── 古诗/    13 首（课内 5 + 课外诵读 8）

历史/
├── 隋唐/    5 段（贞观之治、三省六部、玄奘西行、文成公主、开元盛世）
├── 宋元/    3 段（王安石变法、宋代海上贸易、元代行省制）
└── 明清/    2 段（郑和下西洋、王阳明心学）

道法/
├── 青春/    2 个案例
├── 情绪/    2 个案例
├── 集体/    2 个案例
└── 法治/    2 个案例
```

## 单篇结构

每篇 markdown 包含：

1. **原文 / 案例陈述**（古诗：完整原诗；文言：原文；史料：节选 150-250 字；道法：抽象情境）
2. **字词注释 / 关键术语**
3. **白话翻译 / 现代解读**
4. **时空背景（30 秒了解）**
5. **三问**（理解 / 分析 / 延伸 — 适合家庭饭桌讨论）
6. **彩蛋**（一句话延伸）
7. **同期推荐**（一并读了印象更深）

## 设计理念

- **课内同步**：每篇紧扣人教版七下教材具体单元课次
- **课标素养**：每篇标注对应的 2022 年版课程标准核心素养
- **不刷题、做深度**：不是题库，是经典素材
- **可家庭讨论**：每篇都附亲子开放式问题

## 用途

适合：
- 初中生家庭晚间亲子共读
- 自主深度学习
- 教师备课参考
- 与每日学习简报 Agent 集成（每天抽 3 篇推送到家庭群）
